🎮 Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and track your score in real-time.

 Features
 Play Rock, Paper, Scissors against computer
 Random computer move generation
 Live score tracking (Wins, Losses, Tie)
 Score saved using localStorage
 Reset score functionality
 Clean modern UI with hover effects
 Technologies Used
 HTML
 CSS
 JavaScript

Click on Rock, Paper, or Scissors
Computer will randomly choose a move
Result will be displayed instantly
Score updates automatically
Click Reset to restart the game
